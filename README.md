>인용
- "#"으로 제목과 부제목 등을 설정이 가능
- 순서가 있는 목차와 없는 목차 설정이 가능
- 백텍을 사용하여 코드 입력도 가능
``` python

print("a+b")
```

## 깃허브에 올리는 법
- 작성 후 저장할 것 (ctrl + s)
- git init 입력
  - 폴더당 초반에 한번만 입력해야함
  - 두번 이상 입력하게되면 절대 안됨
- 저장 후 터미널에서 git add .
- git commit -m "설명작성"
